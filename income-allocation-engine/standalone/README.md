# Net Worth Simulator - Standalone Version

This is a standalone HTML file that contains the complete Net Worth Simulator with Income & Savings Allocation engines.

## How to Use

1. **Simply open the file:**
   - Double-click `net-worth-interactive.html` to open it in your default browser
   - Or right-click → "Open with" → choose your browser

2. **Or run a local server:**
   ```bash
   # From the project root:
   npm run serve
   # Then visit: http://localhost:3000/net-worth-interactive.html
   ```

## Features

- Interactive 40-year net worth projection
- Income Allocation Engine (Needs/Wants/Savings)
- Savings Allocation Engine (EF, Debt, Retirement, Brokerage)
- Scenario comparison
- Quarterly allocation table
- Crypto-style chart visualization

## Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- No installation needed - all dependencies are loaded via CDN
